# Summary

This is a boat dock designed to complement any boat models out there.  I designed it to accommodate the benchy tugboat, but scaling will be necessary depending on the size of the tugboat.

If you like this but need more room for more boats, check out my larger boat dock here! https://www.thingiverse.com/thing:3608109

The benchies featured in the photo are from https://www.thingiverse.com/thing:763622.

Note:  I highly recommended printing this with supports enabled.  There are two overhangs which are not bridges in this part.
